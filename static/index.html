Flexbox Magic


The main idea behind the flex layout is to give the container the ability to alter its items' width/height (and order) to best fill the available space (mostly to accommodate to all kind of display devices and screen sizes). A flex container expands items to fill available free space, or shrinks them to prevent overflow.

How flexbox works?

Flexbox consists in a parent container (the container-flex) and their immediate children (flex-items). We always have to assign the property display:flex or display:inline-flex to the parent container in order to use flexbox. Automatically the children become flex-items.

Flex-container properties

Flex direction


Defines the direction of the flex-items. The values can be: row, row-reverse, column and column-reverse

Flex wrap


The flex container align the items in one line by default without make a line break. The flex-wrap propiety takes sense in order to define if we want to break the line and in which direction. The values can be: nowrap, wrap or wrap-reverse

Flex flow


Is the result of the flex-direction and the flex-wrap. We have to put the both values there.

Justify content


Distributes the items for the direction. The values are: flex-start (align the items at the begging of the container), flex-end (align the items at the end og the container), center (align the items at the middle of the container), space-between (leaves spaces between items arranging the first and the last at the container) and space-around (leave spaces between items includes the first one and the last one)

Align content


Align the items horizontally (based on the horizontal axis). The values are: stretch (makes the items take the remaining space), flex-start (align the items at the top of the container), flex-end (align the items at the bottom of the container), center ( align the items at the middle of the container ), space-between (distribute the lines between space. the first line at the start and the last at the end) and space around (distributing each line with equal space)

Order


Indicates the order in which each of the flex items will appear on the flex-container

Flex grow


Indicates the proportion in which items grow as they have free space. If there is only one value will grow all alike.

Flex shrink


It is the opposite of flex-grow. Distribute the size of the items if space is missing.

Flex basis


Defines the default size of an element before the remaining space is distributed.

Flex


It is a shorthand for the flex-grow, flex-shrink and flex-basis attributes

Align self


Overwrites the align-items property for the item in question. The values are the same as the align-items property.
